вес и суммарный объем результата.

Реализован безопасный алгоритм восстановления пути, который проверяет достижимость состояний перед выводом списка артефактов. Код разделен на заголовочный файл и файл реализации для соответствия профессиональным стандартам разработки.

вывести единственное число : минимальную сумму, а во второй строке вывести путь в виде строки символов, обозначив символом L движение влево, а символом D - движение вниз.

Добавлена функция валидации размера игрового поля (ограничение диапазона от 1 до 80). Реализована проверка каждого значения в матрице на соответствие допустимому диапазону. Код приведен к стандарту PascalCase, добавлена обработка ошибок открытия файлов с выводом уведомлений в консоль.
